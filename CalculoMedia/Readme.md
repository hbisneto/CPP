# Cálculo de Média

O programa verifica se o aluno foi aprovado ou reprovado de acordo com alguns critérios:

- Nota da prova 1;
- Nota da prova 2;
- Nota da PMU (Prova Multidisciplinar Unificada);
- Listas de Exercícios;
- Artigo;
- Média;
- Média da Instituição;
